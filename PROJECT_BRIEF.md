# PAGMAR - Project Brief ("שמיים שנפתחים")

## Core Concept
A poetic interactive web experience. Not a horoscope. Not a map you receive.

Core sentence: "Not a map you receive — but skies that open."
Second sentence: "Meaning is not given immediately. It grows through looking."

## The Essential Rule
The sky must NOT reveal itself on load.
The map begins almost empty — dormant dark points, barely visible.
Only through movement, zooming and time near a point does meaning emerge.

## Experience Flow
1. Opening screen (dark, slow, one sentence)
2. Questionnaire (8 poetic questions — creates the hidden seed of the map)
3. Sky Map (dark dormant field → progressive reveal through interaction)
4. Horizon Screen (personal summary, emerges at the end)

## Interaction
- Mouse drag = pan through the sky world
- Mouse wheel = zoom in (approach meaning) / zoom out (see wider horizon)
- Scroll inside sky must NOT scroll the page
- Proximity + time near a point triggers reveal stages
- Nothing reveals instantly

## Build Order
Vertical slices only. Do not build the whole project at once.
Current slice: The core reveal system — dormant points → 6-stage progressive revelation.
